Since strava only provides fitness for cycling. Here you will be able to view your fitness trend using all activities where you held a heart rate monitor and/or cycling power meter and/or swimming activities having distance & time. It's an improved feature which where missing, not a duplicate! Hoping that strava will make it for all sports!

warning Only activities having heart rate data and/or cycling power meter data and/or swimming activities having distance & time will be used. You will not see others activities at the moment. It's planned by the way (e.g. Run or cycling only with the strava app)
The multi sports fitness trend is based on an impulse-response model by Dr. Eric W. Banister in 1975 and Andrew R. Coggan later. The following reference has been used: Modeling Human Performance, fellrnr.com.
